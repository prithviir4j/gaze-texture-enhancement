\# Gaze-Based Texture Enhancement for Video



This project demonstrates gaze-driven texture enhancement for video playback.



\## Features

\- Video upload through browser

\- Real-time gaze detection using Face API

\- Region-of-interest texture enhancement

\- Webcam-based gaze prediction

\- Canvas rendering for dynamic enhancement



\## Technologies Used

\- JavaScript

\- HTML5 Canvas

\- Face-api.js

\- CSS



\## How it Works

1\. Webcam captures user's face

2\. Face-api detects eye landmarks

3\. Gaze coordinates are estimated

4\. Region around gaze is enhanced in the video



\## Usage

1\. Upload a video

2\. Allow webcam access

3\. Look at different parts of the video

4\. The region you look at becomes sharper
